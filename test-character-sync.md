## 🐛 トラブルシューティング

### エラー: "Cannot read properties of undefined (reading 'syncMode')"

**原因**: video-data-master.json の構造が不正

**解決策**: 
- `public/video-data-master.json` が正しく作成されているか確認
- `metadata.syncMode` が "character-level" に設定されているか確認

### エラー: "Module not found"

**原因**: 依存関係が不足

**解決策**:
```bash
npm install
```

### プレビューが表示されない

**原因**: Remotion Studioが起動していない

**解決策**:
```bash
npm run dev
```
