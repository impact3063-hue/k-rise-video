## 🔧 技術的詳細

### アルゴリズム設計

```
優先度1: 句読点（「、」「。」「！」「？」）
         ↓ 最優先で分割
優先度2: 文節境界（名詞、動詞、形容詞などの自立語の前）
         ↓ max_chars以内で分割
優先度3: 最大文字数制限（max_chars = 14）
         ↓ 超過時は緊急分割
```

### データフロー

```
1. Whisper API → 単語レベルタイムスタンプ
2. calculate_character_timestamps() → 1文字単位タイムスタンプ
3. detect_phrase_boundaries() → 文節境界検出（Janome）
4. segment_text_by_phrase() → 文節考慮セグメント分割
5. build_karaoke_subtitles() → 字幕オブジェクト生成
6. prevent_overlap() → 重なり防止
7. JSON出力 → public/video-data-master.json
```
