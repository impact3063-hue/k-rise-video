## 🎯 カスタマイズポイント

### 1. タイミング調整
音声に合わせて `startFrame` と `endFrame` を調整：
```json
{
  "text": "テキスト",
  "startFrame": 0,    // 開始フレーム（30fps）
  "endFrame": 89,     // 終了フレーム
  "style": "normal"
}
```

### 2. テキスト変更
`text` フィールドを編集（`\n` で改行可能）：
```json
{
  "text": "1行目\n2行目\n3行目"
}
```

### 3. スタイル変更
`style` を変更：
- `"normal"` - 通常（黄色）
- `"bullet"` - 箇条書き（金色）
- `"cta"` - CTA（ピンク）

## 💡 Tips

### タイミング計算
- **1秒 = 30フレーム**
- 3秒 = 90フレーム
- 0.5秒 = 15フレーム

### フレーム範囲の例
```
0〜3秒   → startFrame: 0,   endFrame: 89
3〜7秒   → startFrame: 90,  endFrame: 209
7〜9秒   → startFrame: 210, endFrame: 269
```

## 🔧 トラブルシューティング

### Q: テキストが表示されない
A: フレーム範囲を確認してください

### Q: 音声が再生されない
A: `public/` フォルダに音声ファイルがあるか確認

### Q: スタイルが反映されない
A: `style` の値が正しいか確認（"normal", "bullet", "cta"）
