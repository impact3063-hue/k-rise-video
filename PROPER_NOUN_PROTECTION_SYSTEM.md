# 🎯 固有名詞・文脈的表現 完全分離システム v3.2
## Proper Noun and Contextual Expression Separation System

---

## 📋 概要

このシステムは、動画テキストパース処理において**固有名詞（K-RISE、BTS等）**と**文脈的表現（「とは」等）**を完全に分離し、不自然な分割や誤認識を防止します。

### 🎯 解決する課題

| 課題 | 従来の問題 | v3.2の解決策 |
|------|-----------|-------------|
| 固有名詞の分割 | 「K-RISE」が「K」「-」「R」「I」「S」「E」に分割される | 固有名詞辞書により一塊として認識・保護 |
| 文脈的表現の混在 | 「K-RISEとは」で「とは」が固有名詞の一部と誤認識 | 文脈的表現を独立して識別・分離 |
| セグメント分割の不自然さ | 固有名詞の途中で字幕が分割される | 固有名詞の境界を尊重した分割 |

---

```json
{

    "definitions": ["とは", "というのは", "って何"],

```

### 2. **proper_noun_parser.py**
固有名詞パーサーの実装

**主要クラス:**
- `ProperNounParser`: 固有名詞と文脈的表現を識別・分離

**主要メソッド:**
- `identify_proper_nouns_in_text(text)`: テキスト内の固有名詞を識別
- `identify_contextual_expressions(text, proper_noun_positions)`: 文脈的表現を識別
- `parse_text_with_boundaries(text)`: トークン分割
- `annotate_character_data(character_data, text)`: 文字データにアノテーション付与
- `get_segmentation_boundaries(text, respect_proper_nouns=True)`: セグメント境界取得

### 3. **generate_video_data_master_v3.2.py**
固有名詞保護機能を統合した動画データ生成スクリプト

**新機能:**
- 固有名詞パーサーの統合
- 固有名詞を尊重した文節境界検出
- 固有名詞の内部での分割を完全防止

### 4. **test_proper_noun_system.py**
テストスイート

**テストケース:**

## 🚀 使用方法

### 基本的な使用例

```python
from proper_noun_parser import ProperNounParser

# パーサーの初期化
parser = ProperNounParser()

# テキストの解析
text = "K-RISEとは、アジアモデルフェスティバルです。"

# 固有名詞を識別
proper_nouns = parser.identify_proper_nouns_in_text(text)
# 結果: [{"noun": "K-RISE", "start": 0, "end": 6}, ...]

# 文脈的表現を識別
contextual = parser.identify_contextual_expressions(text, proper_nouns)
# 結果: [{"expression": "とは", "start": 6, "end": 8}, ...]

# トークン分割
tokens = parser.parse_text_with_boundaries(text)
# 結果: [
#   {"text": "K-RISE", "type": "proper_noun"},
#   {"text": "とは", "type": "contextual_expression"},
#   ...
# ]
```

### 動画データ生成での使用

```bash
# v3.2スクリプトを使用
python generate_video_data_master_v3.2.py
```

**処理フロー:**
1. 固有名詞パーサーを初期化
2. 音声生成（OpenAI TTS）
3. 音声解析（Whisper API）
4. 固有名詞を識別
5. 1文字単位のタイムスタンプ計算
6. 固有名詞を保護しながらセグメント分割
7. JSON出力

## 📊 処理の詳細

### 1. 固有名詞の識別

**アルゴリズム:**
1. 辞書から固有名詞リストを読み込み
2. 長い順にソート（部分一致を防ぐため）
3. テキスト内のすべての出現位置を検索
4. 重複を除去（長い方を優先）

**例:**
```
入力: "K-RISEとは、K-POPオーディション"
出力: [
  {"noun": "K-RISE", "start": 0, "end": 6},
  {"noun": "K-POP", "start": 10, "end": 15}
]
```

### 2. 文脈的表現の識別

**アルゴリズム:**
1. 固有名詞の位置情報を受け取る
2. 文脈的表現を検索
3. 固有名詞の内部でないことを確認
4. 位置情報を返す

**例:**
```
入力: "K-RISEとは" + 固有名詞位置[0-6]
出力: [{"expression": "とは", "start": 6, "end": 8}]
```

### 3. セグメント境界の決定

**優先順位:**
1. **固有名詞の境界**（最優先）
2. 句読点（「、」「。」等）
3. 形態素解析による文節境界
4. 助詞の位置

**重要な原則:**
- 固有名詞の内部では**絶対に**分割しない
- 固有名詞の直後は分割可能な境界として扱う

## 🔍 トラブルシューティング

### 問題: 固有名詞が検出されない

**原因:** 辞書に登録されていない

**解決策:**
1. [`proper_nouns_dictionary.json`](proper_nouns_dictionary.json) を開く
2. 該当カテゴリに固有名詞を追加
3. スクリプトを再実行

### 問題: 固有名詞が分割される

**原因:** パーサーが初期化されていない

**解決策:**
```python
# v3.2スクリプトを使用していることを確認
python generate_video_data_master_v3.2.py
```

### 問題: 「とは」が固有名詞の一部として認識される

**原因:** 文脈的表現の識別が機能していない

**解決策:**
1. [`proper_nouns_dictionary.json`](proper_nouns_dictionary.json) の `contextualExpressions` を確認
2. 「とは」が登録されていることを確認
